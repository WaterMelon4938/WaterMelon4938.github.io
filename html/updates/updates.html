Update History

0.1.0
0.1.0 was the first release for the New Nintendo 3DS Edition which came out on September 13/14, 2017. It cost $29.99 at launch. It is one of the most uncommon and low quality versions of Minecraft ever. This is due to the 3DS being a very low quality console when it comes to power.
1.1.06
1.1.06 was an update for the New Nintendo 3DS Edition released on November 8/9, 2017, which improved performance and fixed bugs.
1.2.12
1.2.12 was an update for the New Nintendo 3DS Edition released on December 22, 2017, which added achievements.
1.3.12
1.3.12 was an update for the New Nintendo 3DS Edition which added the features of the Boss Update. This update was released on January 30/31, 2018, and added ocean monuments.

1.6.11 was an update for the New Nintendo 3DS Edition released on July 24/25, 2018, which added new DLC packs to the store.
1.7.10
1.7.10 was an update for the New Nintendo 3DS Edition released on September 19/20, 2018, which added the features of the Ender Update, which included the End along with all End-related content (excluding shulker boxes). This is the first version to be released outside of North America and Japan.
1.8.13
1.8.13 was an update for the New Nintendo 3DS Edition released on November 20/21, 2018, which added new downloadable content and fixed bugs.
